# Design System

## Design Philosophy

The website should feel:

- Modern
- Premium
- Minimal
- Trustworthy
- Fast

Inspired by:

- Stripe
- Vercel
- Linear
- Notion

---

## Color Palette

Primary:
Blue (#2563EB)

Background:
White (#FFFFFF)

Secondary Background:
Light Gray (#F8FAFC)

Text:
Dark Gray (#111827)

Success:
Green (#16A34A)

Warning:
Amber (#F59E0B)

Error:
Red (#DC2626)

---

## Border Radius

Cards: 16px

Buttons: 12px

Inputs: 12px

---

## Shadows

Very subtle.

Never use heavy shadows.

---

## Typography

Font:
Geist (default Next.js font)

Headings:
Bold

Body:
Regular

Readable line height.

---

## Buttons

Large

Rounded

Consistent height

One primary style

One secondary style

---

## Cards

White background

Rounded corners

Subtle border

Light shadow

Hover effect:
Small elevation only

---

## Icons

Use Lucide React icons.

Keep icon style consistent.

---

## Spacing

Consistent spacing throughout.

Use Tailwind spacing scale.

Avoid cramped layouts.

---

## Mobile First

Every page should work perfectly on mobile before desktop.

---

## Accessibility

Keyboard accessible.

Visible focus states.

Proper contrast.

Semantic HTML.

---

## Performance

Optimize images.

Avoid unnecessary JavaScript.

Keep Lighthouse score above 95.
